This repository is for MR-based design tools created at DMF Lab, University of Bristol, UK

It has two projects:

**D2N** - This lets you perform low-fidelity CFD simulations for air flow over a drone wing. You can also edit the underlying geometry by moving control points and visualising its impact on flow patterns.

**Pro2** - This project demonstrates several analysis-based design tools.  
**HospitalRoom_Hinge_based_kinematic** shows a hospital light assembly that can simulate the full kinematic motion of the light assembly. By grabbing and moving the spotlight, the user can explore the accessible regions within the workspace.  
**HospitalRoom_Rot_transformers** is also a hospital light assembly, but allows user to move individual links as per kinematic constraints. It also allows changing the size of a few links. In addition, there is an assembly space where users can create a hospital light assembly by bringing links together and putting a pin inside to join the links via rotational motion.  
**PPT_CAD_and_Pneumatics** gives two different analysis-based design tools. **First** provides two activities: _Falling spheres_: Arrange cubes so as to touch all target spheres using falling spheres, and _Cam design_: Desing a cam such that Ram continuously touches moving boxes (indicated by green). The user can change the shape of the cam and the speed of the cam. The **second** provides a tool set to design pneumatic circuits.
